# Medical Appointments System

A fullstack web application designed for managing medical appointments and patient records. Built with modern technologies, this system provides a robust RESTful API backend and a responsive, server-side rendered frontend for seamless user experience.

## üåü Features

- **Patient Management**: Create, update, and maintain detailed patient records with personal and medical information.
- **Appointment Scheduling**: Efficiently schedule, reschedule, and track medical appointments with real-time availability.
- **User Authentication**: Secure authentication system using Laravel Sanctum for API token management.
- **RESTful API**: Well-documented API endpoints for seamless integration and data management.
- **Responsive Design**: Fully responsive Angular frontend that works flawlessly on desktop, tablet, and mobile devices.
- **Server-Side Rendering**: Enhanced performance and SEO with Angular SSR capabilities.
- **Database Management**: Robust MySQL database with proper migrations and relationships between entities.

## üéØ Project Structure

The project is organized into two main directories:

- **`frontend/`**: Angular application with SSR configuration, components, services, and routing.
- **`backend/`**: Laravel API with controllers, models, migrations, and RESTful endpoints.

Both parts work together to provide a complete fullstack solution with clear separation of concerns.
